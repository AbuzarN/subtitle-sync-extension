Hello, this is a scuffed vibecoded extension used to load my own subtitles while watching a video.
My usecase:
  I wanted to watch a show in a different language but there does not exist a website that has the correct dub and sub in the languages I want.
  This issues with Loading SRT files to websites is during midroll ads the srts would be out of sync so I got tired to manually shifting them
  This tool lets me add or subtract delays from subtitles.
